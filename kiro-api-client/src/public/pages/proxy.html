未启用
HTTP 代理配置
配置代理服务器用于刷新 Token、获取用量、对话等 API 请求
启用代理
开启后，所有 API 请求将通过代理服务器发送
支持格式：host:port:username:password (ISP格式) 或 http://username:password@host:port
使用说明
代理配置的使用方法和注意事项

代理将应用于以下操作：

  • 刷新 Token (Social/Builder ID/IdC)
  • 获取账号用量信息
  • 对话请求 (流式和非流式)
  • 获取可用模型列表

ISP 代理格式示例：

38.111.61.65:443:NNDyNdwnJkFt:ykWE3acywd

标准 HTTP 代理格式：

http://username:password@proxy.example.com:8080
